# Field — Fitness App

## Structure
- `athlete/` — Mobile athlete-facing app (4 screens: Home, Workout, Log, Progress)
- `coach/` — Desktop coach admin portal (Dashboard, Roster, Program Builder)
- `sessions/` — Individual workout execution screens

## Tech
Currently plain HTML/CSS/JS. No framework, no build step.
Fonts loaded from Google Fonts (Barlow, DM Mono, Instrument Serif).

## Intent
Migrate to a proper React/Next.js app. Each HTML file is a fully working
prototype with all state and logic inline — use these as the design and
interaction source of truth.
